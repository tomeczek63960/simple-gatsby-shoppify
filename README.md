## Technologies
* Front-end
  * Gatsby
  * GraphQl
  * Styled component

* Back-end
  * Shopify
  
## Features

* Fetching products from shopify
* Adding/Removing product from cart
* Viewing full details of products
* Responsive Design
* Filtering products by name/categories
